<h1>Blog on angular</h1>
<h2>Install</h2>
<pre>
<code>
$ npm install  
$ bower install jquery angular angular-resource angular-route bootstrap jquery-ui
</code>
</pre>

<h2>Start</h2>
<pre>
<code>        
$ gulp
</code>
or
<code>        
$ gulp default
</code>
</pre>

<h2>Open links</h2>

<pre>
<code>        
http://localhost:9000/
http://localhost:3000/ (json server )
</code>
</pre>
